REDES TCP/IP

Una red TCP/IP se compone básicamente de los siguientes componentes:

  1. Cableado de red categoria 7a
  2. Armario de cableado o rack
  3. Conmutadores Gigabit Ethernet
  4. Tarjetas de red NIC en las estaciones de red

GLOSARIO

Conceptos importantes en este módulo son:

HMTL.
Es un lenguaje de etiquetado o de marcado que se utiliza para el desarrollo y confección de páginas web en Internet. Son las siglas de HyperText Markup Language o Lenguaje de Marcado de Hipertexto. Está basado en SGML.
SGML.
Son las siglas de Standard Generalized Markup Language o lenguaje de marcado generalizado estándar. Es el estándar ISO 8879 del año1986 que establece lenguajes de marcado para documentos de forma generalizada de forma declarativa y rigurosa.
XHTML.
Son las siglas de eXtensible Hyper Text Markup Language o lenguaje extensible de marcado de hipertexto. Establece una sintaxis y unas normas rigurosas para el código HTML para que éste sea expresado como XML válido.
XML.
Son las siglas de eXtensible Markup Language o lenguaje de marcado extensible o lenguaje de marcas extensible. Es un meta-lenguaje desarrollado por el W3C que permite crear y definir otros lenguajes de marcas.
