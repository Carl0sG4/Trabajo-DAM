Mapa de Comparación y Contraste

Realidad Virtual (RV) vs. Realidad Aumentada (RA)

REALIDAD VIRTUAL (RV)

DEFINICIÓN PRINCIPAL

Entorno 100% digital. Generado por informática que reemplaza por completo al mundo real.

RELACIÓN CON EL ENTORNO FÍSICO

Sustitución: El usuario se desconecta del mundo físico. Inmersión total.

DISPOSITIVOS DE ACCESO

Requiere hardware específico de aislamiento como cascos de RV (visores cerrados) y guantes con sensores.

TIPO DE EXPERIENCIA

Simulación de entornos (reales o imaginarios) para "transportar" al usuario a otro lugar.

REALIDAD AUMENTADA (RA)

DEFINICIÓN PRINCIPAL

Superposición de capas digitales (texto, imagen, vídeo) sobre el entorno real, enriqueciéndolo.

RELACIÓN CON EL ENTORNO FÍSICO

Enriquecimiento: El usuario mantiene el contacto visual con el mundo físico; este sigue siendo visible.

DISPOSITIVOS DE ACCESO

Accesible mediante dispositivos cotidianos como teléfonos inteligentes, tabletas o gafas especializadas transparentes.

TIPO DE EXPERIENCIA

Interacción en el lugar presente, añadiendo información útil o gráficos a lo que el usuario ya está viendo.

ELEMENTO CENTRAL (SIMILITUD)

Ambas son Tecnologías Inmersivas que utilizan capacidad computacional avanzada para alterar la percepción humana.

FOCO VISUAL PRINCIPAL

RV: Reemplazar la realidad

RA: Enriquecer la realidad
