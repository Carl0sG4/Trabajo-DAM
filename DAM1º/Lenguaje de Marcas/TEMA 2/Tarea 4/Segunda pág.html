CSS y el artributo style

En esta página veremos que es la etiqueta style, donde normalmente se utiliza y datos curiosos sobre esta:

  • ¿Qué es la etiqueta <style>? Se utiliza en HTML para incluir reglas de CSS directamente dentro del documento. Permite definir colores, fuentes, tamaños, márgenes y más, sin necesidad de un archivo CSS externo.

  • ¿Dónde se utiliza normalmente? Generalmente se coloca dentro de la sección <head> del documento HTML. De esta forma, los estilos se aplican al cargar la página, antes de que se muestre su contenido.

  • Datos curiosos sobre la etiqueta <style>:
    • Puede colocarse también dentro del <body>, aunque no es una práctica recomendada.
    • Admite el uso de selectores de CSS, igual que en los archivos .css externos.
    • Si hay varios bloques <style>, los estilos definidos más abajo pueden sobrescribir los anteriores.
    • Fue introducida oficialmente en HTML 4 para permitir separar contenido y presentación.
