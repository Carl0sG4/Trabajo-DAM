Para utilizar el elemento track, se debe de realizar con un archivo local, puesto que no funciona con videos de youtube. al inverstigar, descubrí, que estos no pueden utilizar esa etiqueta debido a que están dentro de un inframe, un tipo de etiqueta que no permite modificar el contenido. Por último, el lenguaje de marcas con el que esta relacionado el elemento track es WebVTT (Web Video Text Tracks Format), un lenguaje de marca basado en texto y creado especificamente para describir pistas de texto sincronizadas con un video o audio en HTML5.
