Archivo 2: Condicionales y Bucles

1. Clasificación por Edad

Uso de if/else para categorizar una entrada numérica.

// Código Original:
let edad = Number(prompt("Tu edad: "));

if (edad <=12) console.log("Niño");
else if (edad <=17) console.log("Adolescente");
else if (edad <= 64) console.log("Adulto");
else console.log("Jubilado");

2. Contador Ascendente/Descendente

Bucle For que se adapta según si el número es positivo o negativo.

// Código Original:
let numero = Number(prompt("Un número: "));

if (numero > 0) {
    for(let i = 1; i <= numero; i++) {
        console.log(i);
    }
} else {
    for(let i = 0; i >= numero; i--) {
        console.log(i);
    }
}
